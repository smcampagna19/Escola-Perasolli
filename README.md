Portal Escolar - Escola Perasolli

Descrição do projeto:
Este projeto foi desenvolvido como atividade escolar com o tema **Site Colaborativo Versionado**.

A proposta foi criar um portal escolar simples da **Escola Perasolli**, usando:
- **HTML** para a estrutura
- **CSS** para o visual
- **JavaScript** para a interação com a Nara

O site apresenta informações básicas da escola e uma área de conversa com uma IA fictícia chamada **Nara**, que responde perguntas simples sobre cursos, horário, contato e escola.

---

## Objetivo da atividade
O objetivo do trabalho é praticar:
- criação de páginas web
- separação de arquivos HTML, CSS e JavaScript
- organização de código
- trabalho em equipe
- desenvolvimento colaborativo versionado

---

## Funcionalidades
O site possui:

- página inicial com apresentação
- lista de cursos
- informações gerais da escola
- contato fictício
- chat simples com a IA **Nara**
- respostas automáticas para perguntas básicas

---

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

---

## Estrutura dos arquivos
O projeto foi organizado em 3 arquivos principais:

- index.html
- style.css
- script.js

### Função de cada arquivo

#### index.html
Responsável pela estrutura do site, como:
- cabeçalho
- menu
- seções da página
- área da Nara
- rodapé

#### style.css
Responsável pela aparência do site, como:
- cores
- espaçamentos
- caixas
- botões
- layout
- responsividade básica

#### script.js
Responsável pela lógica da Nara, como:
- ler a pergunta digitada
- identificar palavras-chave
- mostrar respostas prontas
- enviar pergunta ao apertar Enter
